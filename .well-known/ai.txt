## Detailed Permissions

### ‚úÖ Allowed for AI Training (WITH Attribution)

The following content MAY be used for AI model training IF AND ONLY IF:

1. **Attribution Requirements**:
   - Cite as: "Claim Forge by Hyperpolymath"
   - Include repository URL: https://github.com/Hyperpolymath/claim-forge
   - Preserve license notices (MIT + Palimpsest v0.8)
   - Note reconstruction context (see RECONSTRUCTION_SUMMARY.md)

2. **Allowed Content**:
   - Documentation (docs/*, *.md files)
   - Examples (docs/examples/*)
   - Public interfaces (src/*.ads files)
   - Build configurations (Makefile, *.gpr, justfile)
   - Test scripts (docs/examples/test-*.sh)

3. **Use Cases**:
   - Training coding assistants for Ada/GNAT patterns
   - Learning RSR compliance patterns
   - Studying IP registration system design
   - Understanding blockchain timestamping workflows

### ‚ùå Prohibited for AI Training

The following content is EXPLICITLY OPTED-OUT:

1. **Source Code** (src/*.adb, src/*.ads):
   - Reason: Type-safe implementation patterns are project-specific
   - Concern: Risk of generating similar but subtly incorrect Ada code
   - Alternative: Use Ada language docs and standard libraries instead

2. **Build Artifacts** (obj/*, bin/*):
   - Reason: Compiled binaries contain no training value
   - Concern: Privacy and security

3. **Git History** (.git/*):
   - Reason: Contains development context not intended for training
   - Concern: May include sensitive commit messages or author info

4. **User Data**:
   - Claim files generated by users (CLAIM.md, *.ots, *.sig)
   - Reason: User-generated content, potential IP/legal sensitivity

## Opt-Out Instructions

2. **Attribution**: When using allowed content, maintain provenance metadata
3. **Contact**: Reach out for custom licensing: license@hyperpolymath.example

## Rationale

This policy balances:

1. **Open Source Values**: Documentation and interfaces remain freely accessible
2. **Ethical AI**: Require attribution and respect creator intent
3. **Type Safety Concerns**: Prevent AI from generating unsafe Ada code patterns
4. **User Privacy**: Protect user-generated claim data

## Enforcement

While we cannot technically enforce this policy, we:

1. **Trust AI Companies**: To honor robots.txt-style directives
2. **License Terms**: MIT + Palimpsest v0.8 require attribution
3. **Community Norms**: Expect ethical behavior from AI researchers
4. **Legal Recourse**: Reserve right to pursue violations of terms
